ViqueiraQRNN.circuit.CircuitQRNN

class ViqueiraQRNN.circuit.CircuitQRNN(nE: int, nM: int, nT: int, repeat_encode: int, repeat_evolution: int, ansatz: AnsatzQRNN, init_state_mem: cunqa.circuit.CunqaCircuit | None = None)

CircuitQRNN.__init__(nE: int, nM: int, nT: int, repeat_encode: int, repeat_evolution: int, ansatz: AnsatzQRNN, init_state_mem: cunqa.circuit.CunqaCircuit | None = None)

Class to manage a QRNN circuit. This circuit modifies a time series to obtain another time series after executing. The default ansatz is the one from the Exchange-Memory w Controlled Z-gates algorithm (https://arxiv.org/abs/2310.20671).

Parameters:

  • nE (int) – number of qubits for the Environment/Exchange register

  • nM (int) – number of qubits for the Memory register

  • nT (int) – number of time steps of the time series

  • repeat_encode (int) – number of times that the encoding block should be repeated in the circuit

  • repeat_evolution (int) – number of times that the evolution block should be repeated in the circuit

  • ansatz (int) – AnsatzQRNN instance to apply on each time step

  • init_state_mem (<class CunqaCircuit>) – initial state for the memory register

Returns:

circuit implementing the QRNN structure

Return type:

circuit (<class CunqaCircuit>)

CircuitQRNN.assign_emcz_parameters(new_x: numpy.array, new_theta: numpy.array) → list[float | int]

Method for combining the data from the time series and the theta parameters to update the circuit.

Parameters:

  • theta (numpy.array) – Trainable parameters for encoding and evolution unitaries. Vector lenght: 2nE*repeat_encode + 2(nE + nM)*repeat_evolution + nE.

  • x (numpy.array) – Input data representing a time series. Its shape must be (nT, nE)

Returns:

parameters to insert on the circuit organized in the right order

Return type:

all_params (list[float, int])
